Projetos de Ciência de Dados

Construção de uma modelo de regressão para prever as vendas com 6 semanas de antecedência.

Eu usei Python, estatística e técnicas não-supervisionadas de Machine Learning para construir um modelo para prever o valor de vendas das próximas 6 semanas, para uma rede de farmácias Européia.

Foram aplicados 5 modelos de machine Learning, 3 lineares e 2 não lineares, e escolhido o que obteve melhor resultado, nesse caso o XGBoost.

O modelo treinado obteve 90% de MAPE e foi colocado em produção usando Cloud Computing.

Criei um bot no Telegram onde qualquer pessoa consegue acessar e solicitar a previsão de vendas de qualquer uma das 1015 lojas da rede, informando o ID das mesma.

  • Python, Pandas, NumPy, Flask(criação de API), Matplotlib e Seaborn
